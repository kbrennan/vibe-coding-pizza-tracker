# Vibe Coding Pizza Party Tracker

A beautiful, modern web app to track pizza slices at the final session of Vibe Coding!

## Features

- Real-time slice tracking with +/- buttons
- Visual progress bars for each pizza type
- Live statistics (total remaining, slices eaten, percentage)
- Auto-saves to browser local storage
- Mobile responsive design
- Reset functionality

## Pizza Menu

1. Classic Cheese (48 slices)
2. Margherita Pizza (48 slices)
3. Santorini (24 slices)
4. Pepperoni (64 slices)
5. CBR (32 slices)
6. Hawaiian (24 slices)
7. Vegan (TBD)
8. Gluten-Free (TBD)
9. Dairy-Free & Gluten-Free (TBD)

## How to Use

Simply open the page and click the - button each time someone takes a slice!

## Local Development

Open `index.html` in any modern web browser.
